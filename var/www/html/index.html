Webserv

Custom HTTP server implementation.
This page exposes the available endpoints configured on the server.

Available routes

  • /api
    GET / POST / DELETE API endpoint.
    Go to /api
  • /downloads
    Directory listing (autoindex enabled).
    Browse downloads
  • /upload
    File upload endpoint (POST only).
  • /docs
    Server documentation.
    Open documentation
  • /cgi-test
    CGI execution endpoint (Python).
  • /redir
    HTTP redirection test.
    Test redirect
  • /hidden
    External redirection (301).

Error pages

The server is configured with custom error pages:

  • 400 Bad Request
  • 403 Forbidden
  • 404 Not Found
  • 405 Method Not Allowed
  • 408 Request Timeout
  • 500 Internal Server Error
  • 505 HTTP Version Not Supported
